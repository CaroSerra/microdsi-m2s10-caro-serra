Flujo recomendado

  1. Identifica procesos L1 (verbo + objeto).
  2. Prioriza con Impacto / Esfuerzo / Riesgo.
  3. Define el SIPOC del proceso ganador.

Criterios mínimos

  • Debe tener cliente claro.
  • Debe tener trigger y output definidos.
  • Debe ser medible.
  • Debe tener límites explícitos.
